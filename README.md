# 学生排名系统 (Student Ranking System)

## 项目描述

学生排名系统是一个用Go语言编写的工具，用于读取和处理CSV文件中的学生数据，根据不同的规则对学生进行排序和赋值，并生成结果文件。该系统支持多种选科组合和标识类型，能够灵活处理不同的排序和赋值需求。

## 功能特性

### 1. 数据读取和处理
- 读取CSV文件中的学生数据
- 处理空值和不同格式的数据
- 支持多种选科组合

### 2. 排序规则
- **物理、化学、生物、政治、历史、地理**：先按标识优先级（0 = 西 < 1 < 2），再按总分降序，最后按ID升序
- **语数外**：先按标识优先级（0 < 西 < 1 < 2），再按总分降序，最后按ID升序
- **排名**：按总分降序，不考虑标识顺序

### 3. 赋值规则
- 包含对应关键字的学生：赋值为1、2、3...，超过10时继续递增
- 不包含对应关键字的学生：赋值为"自习X"，从包含关键字的学生数量+1开始

### 4. 结果输出
- 打印学生信息到控制台
- 统计选科组合和标识分布
- 将结果写入result.csv文件

## 技术实现

- 使用Go语言的标准库进行CSV文件读写
- 使用结构体建模学生数据
- 使用自定义排序函数实现复杂的排序逻辑
- 使用映射（map）提高数据查找效率
- 实现了模块化的代码结构，便于维护和扩展

## 使用方法

### 1. 准备数据

确保要处理的CSV文件包含以下列：
- 序号
- 选科
- 考号
- 学生姓名
- 班级
- 总分
- 排名
- 语数外
- 物理
- 化学
- 生物
- 政治
- 历史
- 地理
- 标识

### 2. 配置文件路径

在`main.go`文件中，修改`readCSVFile`函数的参数，指定要处理的CSV文件路径：

```go
// 读取CSV文件
students, err := readCSVFile("test2.csv") // 修改为您的CSV文件路径
```

### 3. 运行程序

```bash
go run main.go
```

### 4. 查看结果

- 控制台输出：显示学生信息、选科组合统计和标识分布
- result.csv文件：包含排序和赋值后的完整结果
